Задание 1. Заполнение двумерных массивов

Алгоритм:

Первый массив (n×m) вводится с клавиатуры, по столбцам, начиная с последней строки.

Второй массив (n×n) заполняется случайными четырёхзначными числами с нечётными цифрами.

Третий массив (n×n) заполняется заранее заданным шаблоном (например, для n=5).

Примеры вывода:

Введите 4 элементов массива A:
1
1
1
1
Матрица A:
         1         1
         1         1

Матрица B:
      2094      3371
      8197       169

Матрица C:
      2094      3371
      8197       169

Задание 2. Работа с двумерными массивами

Алгоритм:

Вычисляется среднее арифметическое первого столбца.

Для каждого следующего столбца подсчитывается количество элементов, превышающих среднее предыдущего столбца.

Примеры вывода:

Элементы результата по индексам:
[0,0] = -33782
[0,1] = 33782
[1,0] = -33782
[1,1] = 33782

Задание 3. Работа с матрицами и перегрузка операторов

Алгоритм:

Определяем выражение 7*A*(Bᵀ - C).

Для этого перегружены операторы * и -.

Для отображения матрицы перегружен метод ToString().

Примеры вывода:

Результат 7*A*(Bᵀ - C):
    -33782     33782
    -33782     33782

Задание 4. Бинарные файлы

Алгоритм:

Создаём бинарный файл с числами.

Находим максимальный и минимальный элемент.

Вычисляем их разность.

Примеры вывода:

Разность max и min элементов бинарного файла:
174

Задание 5. Бинарные файлы и структуры

Алгоритм:

Файл содержит массив структур Toy (название, цена, возраст).

Используется XML-сериализация для записи и чтения файла.

Выводим названия игрушек, цена которых отличается от самой дорогой не более чем на k.

Примеры вывода:

Игрушки из файла:
Мяч - 500 руб.
Робот - 520 руб.
Кубик - 515 руб.

Задание 6. Текстовые файлы (целые числа по одной строке)

Алгоритм:

Создаём текстовый файл с целыми числами.

Подсчитываем количество вхождений максимального элемента.

Примеры вывода:

Количество max элементов в a.txt: 1

Задание 7. Текстовые файлы (несколько чисел в строке)

Алгоритм:

Создаём текстовый файл с числами по несколько в строке.

Подсчитываем количество чётных чисел.

Примеры вывода:

Количество чётных чисел в b.txt: 7

Задание 8. Текстовые файлы (текстовые строки)

Алгоритм:

Создаём текстовый файл с несколькими строками.

Копируем строки, содержащие заданную комбинацию символов, в другой файл.

Примеры вывода:

Строки с 'форма' скопированы в out.txt
